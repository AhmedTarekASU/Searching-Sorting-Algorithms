include Algorithms.h
It provides the Following functions

Binary Search	: it takes address of array of integers, address of variable containing size of array, address of integer to search for
	it returns the index of the element it searched for
	if element is not found, it returns -1

Jump Search	: it takes address of array of integers, address of variable containing size of array, address of integer to search for
	it returns the index of the element it searched for
	if element is not found, it returns -1

Selection Sort	: it takes address of array of integers, address of variable containing size of array
	it sorts the array in ascending order

Insertion Sort	: it takes address of array of integers, address of variable containing size of array
	it sorts the array in ascending order
